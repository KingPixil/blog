Gradient Descent

January 1, 2018

Neural networks work by receiving a set of inputs, multiplying them by a set of weights, and then returning an output. At first, the weights are initialized with random values. Over time, the neural network refines the weights so that it can return a plausible output for any input.

A neural network uses a loss function to find how close a neural network output is to the target output. The goal of the network is to change the weights in such a way that the loss function returns an output close to zero. A low loss value means that the output for all inputs was incredibly close to the expected output for the inputs.

A method called gradient descent is often used to refine ("train") the weights so that the loss value gradually becomes smaller.

Two Variables

Instead of jumping right into a 1000-dimensional gradient descent example, let's start out with a simple function that uses two variables, where xxx is the input, and www is the weight.

f(x,w)=xwf(x, w) = xwf(x,w)=xw

Let's say we want to optimize the weight in a way that gives an output of 444 when given an input of 222. We want f(2,w)=4f(2, w) = 4f(2,w)=4. First, let's start out with a random weight 111. Currently, our function gives us a wrong output.

f(2,1)=2f(2, 1) = 2f(2,1)=2

To help solve this problem, we can define a loss function that takes the output of the function and tells us the error in our output.

l(x,w,t)=t−f(x,w)l(x, w, t) = t - f(x, w)l(x,w,t)=t−f(x,w)

It just gives us the difference between the target output ttt and the output of our function fff given xxx and www. With the initial random weight of 111, we can find the current loss. Remember, we want the output of the function to be 444 when given 222.

l(2,1,4)=4−2=2l(2, 1, 4) = 4 - 2 = 2l(2,1,4)=4−2=2

This makes sense, since increasing the output by 222 would get us our target output 444. Now, we need to find a way to decrease the output of our loss function by changing www. To do this, we need to see how changes in www affect the loss function. Formally, seeing how one variable changes depending on another is called the partial derivative.

Finding derivatives involves using a formula.

Finding partial derivatives involves the same formula, but represents how the output of fff changes as only one variable changes while treating other variables as constants.

We can find the partial derivative of the loss function example with respect to www by substituting the function into the formula.
